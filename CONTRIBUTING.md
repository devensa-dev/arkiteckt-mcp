# Contributing to Architecture MCP

Thank you for your interest in contributing.

This project treats **architecture as enforceable, versioned state**, not documentation.  
Please read this guide before proposing changes.

---

## Getting Started

### Prerequisites

- Node.js (LTS)
- Git
- A basic understanding of MCP and infrastructure-as-code concepts

### Running Locally

```bash
git clone https://github.com/devensa-dev/arkiteckt-mcp.git
cd arkiteckt-mcp
npm install
npm run dev
